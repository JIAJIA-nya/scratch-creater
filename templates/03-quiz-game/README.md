# 问答游戏模板

**难度**：入门
**涉及概念**：变量、条件判断、用户输入、字符串拼接、循环

## 描述

提出 5 个问题，用户输入答案，判断对错并计分，最后显示总分。

## 积木块

- `event_whenflagclicked` — 当绿旗被点击
- `data_setvariableto` — 设置变量
- `data_changevariableby` — 变量增加
- `control_repeat` — 重复 5 次
- `sensing_askandwait` — 询问并等待
- `sensing_answer` — 回答
- `operator_equals` — 等于
- `control_if_else` — 条件判断
- `looks_say` — 显示消息
- `operator_join` — 连接字符串

## 角色要求

- 出题角色：1 个造型
- 背景：任意
